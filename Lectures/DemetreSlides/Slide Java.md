Note: 
- ნატურალურ რიცხვებს ვუწოდებთ თვლის შედეგად მიღებულ **დადებით** მთელ რიცხვთა სიმრავლეს. აღინიშნება N
- მთელი რიცხვები მოიცავს ნატურალურ რიცხვებს, მათ მოპირდაპირე უარყოფით რიცხვებს და ნულს. აღინიშნება Z
- ნამდვილი რიცხვები კი - რაციონალურისა და ირაციონალურების გაერთიანებაა, აღინიშნება დიდი R-ით

---
$$\mathbb{N} = \{ 1, 2, 3, \dots \} $$

$$\mathbb{N}_0 = \{ 0, 1, 2, 3, \dots \} $$

note:
კომპ მეცნიერებაში  არაუარყოფითი მთელი რიცხვების მცნება ძალიან მნიშვნელოვანია. ათვლის წერტილად 0 ხშირად გამოიყენება
- მოდით,  შემოვიღოთ სიმრავლე N_0. 
- რომელიც მოიცავს ნატურალურ რიცხვებს  და 0-ს. 

---
$$\mathbb{N} = \{ 1, 2, 3, \dots \} $$ 

$$\mathbb{N}_0 = \{ 0, 1, 2, 3, \dots \} $$


$$\mathbb{Z} = \{ \dots, -2, -1, 0, 1, 2, \dots \}$$

$$
\mathbb{R} = \mathbb{Q} \cup \mathbb{I}
$$


Note:
ახლა კი, ჩვენი მათემატიკური სიმრავლეების ასე გამოიყურება.
დააკვირდით, რომ თითოეული მათგანი უსასრულო რაოდენობის ელემენტს შეიცავს. ამიტომ მათ
**უსასრულო ზომის სიმრავლეები** ეწოდებათ. 
მიუხედავად მათი უსასრულობისა, მაინც შეგვიძლია მივხვდეთ რომელი უფრო "დიდი" სიმრავლეა. თუ შევხედავთ, ერთი მეორეში მაინც ეტევა, მაგალითად N "დევს" N0-ში და ასე შემდეგ.

---
$$\mathbb{N} \subseteq \mathbb{N}_0 \subseteq \mathbb{Z} \subseteq \mathbb{R}$$

note:
- ამ დროს ვამბობთ, რომ ერთი სიმრავლე მეორის "ქვესიმრავლეა". ეს დამოკიდებულება ამ სიმბოლოთი აღინიშნება.
